The analyst loop

From raw page to verdict in three steps

Open a SIEM alert, a ticket, a CTI report or an email — Mustela keeps the entire investigation inside the browser.

STEP 01

Detect

Mustela scans the page locally and highlights every IOC it finds — without altering the host page's layout or behavior.

STEP 02

Investigate

Click any highlight to open an in-page panel with aggregated provider verdicts and an optional AI summary — no tab switching.

STEP 03

Keep the trace

Verdicts are cached, history is preserved and analyst notes stay attached — all stored locally, exportable as JSON.

Capabilities

Analyst Assist · AI

An AI second opinion, on your terms

Optional and bring-your-own-model. Connect any OpenAI-compatible endpoint — including a fully local model — to turn raw verdicts into decisions.

Investigation summaries

For each IOC, the model weighs verdicts, provider disagreement and page context into a one-sentence assessment and the single best next step.

Generated verdict “High-confidence malicious domain flagged by multiple scanners; recent benign hosting hints at a possible false positive — validate against your DNS logs before escalating.”

Cross-IOC correlation

Ask the model to find patterns across your recent investigations — shared infrastructure, ASN clustering and campaign indicators — with a clear relationship verdict.

Trust posture

Transparent by design

  • Detection runs locally. IOC parsing and highlighting never leave your browser.
  • Storage stays local. Settings, cache, history and notes live in chrome.storage.local.
  • Lookups are explicit. An IOC reaches a provider only when you ask, and only the ones you enabled.
  • AI is opt-in. Bring your own key — or run a local model so nothing leaves your machine.

Full details in PRIVACY_TRANSPARENCY.md.

Deployment

Up and running in two minutes

Works out of the box for detection and pivots. Add free API keys later for enriched verdicts.

RECOMMENDED

Download a release

Download latest release
  1. Unzip mustela-chrome.zip anywhere.
  2. Open chrome://extensions and enable Developer mode.
  3. Click Load unpacked and select the unzipped folder.
  4. The Mustela icon appears in your toolbar — done.

No build tools required.

FROM SOURCE

Build it yourself

# clone, install & build
git clone https://github.com/ArmorIntel/mustela.git
cd mustela
npm install && npm run build

Or run the one-liner (Git + Node 18+):

curl -fsSL https://raw.githubusercontent.com/ArmorIntel/mustela/main/install.sh | bash

Then load dist/chrome via Load unpacked.
