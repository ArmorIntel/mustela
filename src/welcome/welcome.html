Quick start

How the extension behaves

Detect IOC on the page, open details in one click, and keep provider access ready before you need it.

Show usage details ⌄
  1. Open a page containing an IOC: IP, subnet, ASN, domain, URL, or hash.
  2. The extension highlights detectable IOC directly in the page.
  3. Hover an IOC to reveal quick actions and external pivots.
  4. Use the round button to open the investigation details.
  5. Use the popup to triage current-page IOC and reopen recent history.

Trust & transparency

What stays local and what leaves the browser

IOC detection and highlighting run locally in Chrome. Settings, API keys, cache, history, and disabled-page rules are stored in chrome.storage.local on this browser profile.

When you enable VirusTotal, AbuseIPDB, or Shodan lookups, the IOC you investigate is sent to that provider. This MVP does not use its own backend or cloud sync.
